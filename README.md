# 맛나는꼬꼬 · Prasia Lab

- `index.html`: 공개 홈페이지
- `css/style.css`: 전체 디자인

- `data/videos.json`: 영상 목록
- `images/bi_simbol.png`: 상단 심볼
- `images/kkokkoChar.png`: 하단 캐릭터
